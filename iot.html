WHAT IS IoT?

IoT หรือ Internet of Things คือ เทคโนโลยีที่ทำให้อุปกรณ์อิเล็กทรอนิกส์ต่างๆ สามารถเชื่อมโยงหรือส่งข้อมูลถึงกันได้ผ่านอินเทอร์เน็ต โดยไม่ต้องป้อนข้อมูลด้วยมือ ซึ่งช่วยให้สามารถสั่งการควบคุมการใช้งานอุปกรณ์ต่างๆ ผ่านเครือข่ายอินเทอร์เน็ตได้

Environmental monitoring module

อุปกรณ์ตรวจวัดคุณภาพสภาพแวดล้อมและแจ้งเตือนอัตโนมัติ

• ฟาร์มสัตว์ปีก และฟาร์มอัจฉริยะ ช่วยในการควบคุมุสภาพภูมิอากาศด้วยการจัดการ อุุณหภูมิ ความชื้น และการระบายอากาศ
• การเก็บรักษาวัคซีน ตรวจสอบอุุณหภูมิให้เหมาะสมสําหรับการเก็บรักษาวัคซีน ลดความเสี่ยงในการเสื่อมคุณภาพ
• ตู้ฟักไข่/่ห้องฟักไข่ ควบคุมสภาพแวดล้อมให้เหมาะสมสําหรับการฟักไข่เพื่อให้ได้ผลการฟักที่สูง
• ห้องเย็นและคลังสินค้า เหมาะสําหรับสินค้าที่ไวต่ออุณหภูมิ เพื่อให้มั่นใจว่าสินค้านั้นถูกเก็บรักษาในสภาพแวดล้อมที่เหมาะสม
• การตรวจสอบจากระยะไกล การเชื่อมต่อผ่าน Bluetooth และ WiFi ทําให้สามารถตรวจ สอบและควบคุมจากระยะไกลได้สะดวก เหมาะสําหรับการจัดการในพื้นที่ขนาดใหญ่
• ประหยัดพลังงาน แหล่งจ่ายไฟแบบสองทางช่วยให้ทํางานได้ต่อเนื่องแม้ในกรณีที่ไฟดับทำให้ไม่มีการหยุดชะงักในการตรวจสอบและควบคุม

Parking detection (Geomagnetic sensor)

ระบบบริหารจัดการที่จอดรถ (Geomagnetic Sensor)

• Detection by Geomagnetic sensor and algorithm ตรวจสอบสถานะการจอดด้วยคลื่นแม่เหล็กและอัลกอริธึมเฉพาะ
• Indoor/Outdoor Application สามารถประยุกต์ใช้ได้กับทั้งลานจอดรถในและนอกอาคาร
• Robustness, IP68 Waterproof and Dustproof อุปกรณ์ออกแบบให้ได้มาตรฐานป้องกันฝุ่นและน้ำ
• Easy and Robust Mounting อุปกรณ์ติดตั้งง่ายโดยการฝังหรือยึดอุปกรณ์ติดกับพื้นที่ ไม่จำเป็นต้องมีการเดินสาย
• Online checking status via web application สามารถตรวจสอบสถานะลานจอดผ่านเว็ปแอปพลิเคชั่นได้
