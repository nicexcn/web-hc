RTLS TECHNOLOGY

Real-Time Location System

WHAT IS RTLS?

RTLS (Real-Time Locating System) คือ ระบบระบุตำแหน่งแบบเรียลไทม์ เป็นเทคโนโลยีที่ใช้ติดตามและระบุตำแหน่งของบุคคล, วัตถุ, หรือทรัพย์สินได้ทันทีภายในพื้นที่จำกัด โดยอาศัยการส่งสัญญาณระหว่างอุปกรณ์ที่ติดอยู่กับวัตถุ (Tags) กับตัวรับสัญญาณ (Anchors) ในพื้นที่นั้นๆ โดยใช้เทคโนโลยีอย่าง Bluetooth Low Energy (BLE), Wi-Fi, RFID, หรือ UWB เพื่อให้เห็นตำแหน่งที่แน่นอนบนแผนที่ดิจิทัลแบบสดๆ เหมาะสำหรับโรงงาน, โรงพยาบาล, คลังสินค้า เพื่อเพิ่มประสิทธิภาพและความปลอดภัย

Beacon Technology

Beacon คืออุปกรณ์ส่งสัญญาณ Bluetooth Low Energy (BLE) ที่ใช้ส่งข้อมูลหรือสัญญาณไปยังอุปกรณ์ใกล้เคียง เช่น สมาร์ทโฟน หรือแท็บเล็ต โดยไม่ต้องใช้อินเทอร์เน็ตหรือเชื่อมต่อเครือข่ายไร้สาย

การทำงาน Beacon ส่งสัญญาณที่สามารถรับรู้ได้จากอุปกรณ์ที่เปิดใช้งาน Bluetooth ซึ่งจะให้ข้อมูลที่เกี่ยวข้องกับตำแหน่งของผู้ใช้ เช่น โปรโมชั่น, ข้อเสนอพิเศษ หรือข้อมูลนำทาง

ประโยชน์หลัก

• การตลาด: ส่งข้อมูลโฆษณาหรือโปรโมชั่นให้กับลูกค้าหรือผู้ที่อยู่ใกล้ Beacon
• การนำทางภายใน: เช่น ในห้างสรรพสินค้า, สนามบิน, หรือพิพิธภัณฑ์ เพื่อช่วยผู้ใช้ในการหาตำแหน่งหรือเส้นทาง
• การติดตาม: ใช้ติดตามสินค้าหรือทรัพย์สินภายในพื้นที่ เช่น คลังสินค้า

ข้อดี

• ประหยัดพลังงาน: ใช้พลังงานต่ำเนื่องจากเทคโนโลยี BLE
• ไม่ต้องใช้อินเทอร์เน็ต: สามารถทำงานได้โดยไม่ต้องเชื่อมต่อกับเครือข่าย
• ความแม่นยำสูง: สามารถให้ข้อมูลหรือคำแนะนำตามตำแหน่งของผู้ใช้ได้อย่างแม่นยำ

Beacon ทำงานอย่างไร?

อุปกรณ์ในระยะของ Beacon จะรับรู้สัญญาณและสามารถแสดงข้อมูลหรือข้อความเฉพาะตำแหน่งได้ เช่น

• แจ้งโปรโมชั่นเมื่อเดินผ่านร้านค้า
• ช่วยนำทางภายในอาคาร
• ให้ข้อมูลอัตโนมัติในพิพิธภัณฑ์

Beacon มีหลายรูปแบบ เช่น

• แบบติดตั้งถาวรภายในอาคาร
• แบบพกพาขนาดเล็ก
• แบบ USB สำหรับติดตามทรัพย์สิน

ตัวอย่าง สถานที่ต่างๆที่สามารถใช้งาน Beacon

ประเภทสถานที่ วิธีใช้งาน Beacon ตัวอย่าง โปรโมชั่น/Feature
• ร้านกาแฟ / ร้านอาหาร ติด Beacon ใกล้ทางเข้า/ชั้นโปรโมชั่น - แจ้งเตือนส่วนลดเครื่องดื่มเมื่อผู้ใช้เข้าร้าน
- แจกคูปองซื้อ 1 แถม 1
- แจ้งเมนูพิเศษวันนี้
• ร้านค้าปลีก / ร้านสะดวกซื้อ ติด Beacon ในสถานี / รถเมล์ / รถไฟฟ้า - แจ้งเตือนโปรสินค้าใกล้หมด
- สะสมแต้มเพิ่มเฉพาะผู้ใช้ที่อยู่ในร้าน
- แจกคูปองเงินคืน (Cashback)
• ขนส่งสาธารณะ ติด Beacon รอบงาน / บูธ - แจ้งเตือนผู้ใช้เติมเงินบัตร / e-Wallet
- ส่งข้อมูลเส้นทางและเวลารถ
- แจกคูปองส่วนลดค่าเดินทาง
• งาน Event / คอนเสิร์ต / มหกรรม ติด Beacon รอบงาน / บูธ - แจ้งตารางกิจกรรมหรือโชว์พิเศษ
- แจกสิทธิพิเศษ / VIP Pass
- สะสมแต้มแลกของรางวัล
• ร้านบริการ / ศูนย์บริการ / โรคยานาล ติด Beacon บริเวณล็อบบี้หรือจุดบริการ - แจ้งเตือนโปรโมชั่นคอร์ส / แพ็กเกจ
- ส่งข้อความเตือนนัดหมาย
- แจกคูปองส่วนลดเฉพาะผู้มาใช้บริการ
